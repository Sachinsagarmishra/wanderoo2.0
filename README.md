# Wanderoo - Premium Travel Agency Website

Wanderoo is a modern, structured PHP-based website designed for travel agencies. It features a high-end design system, a modular backend architecture, and a dedicated admin panel for content management.

## 🚀 Quick Start
1. **Clone**: `git clone https://github.com/Sachinsagarmishra/wanderoo.git`
2. **Database**: Import `database.sql` into your MySQL server.
3. **Configure**: Update `config.php` with your database credentials.
4. **Login**: Access the admin panel at `/admin/` with:
   - User: `admin`
   - Pass: `admin123`

## ✨ Features
- **Modern UI**: Glassmorphism navigation and responsive hero section.
- **Mixed Typography**: Urbanist & Playfair Display fonts for a premium feel.
- **Admin Panel**: Secure dashboard to manage frontend data.
- **SEO Ready**: Dynamic meta tags and clean HTML structure.
- **PDO Security**: Safe database queries to prevent SQL injection.
